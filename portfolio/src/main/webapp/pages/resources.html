For Those Looking to Begin Web-development:

The Odin Project

If you know that you want to spend a good amount of time becoming a proficient web developer, then this is the comprehensive resource for you. Here, you can learn front-end and back-end development in detail by being challenged with fun, rigorous projects. A benefit of this resource is that it teaches you various ways of setting up a development environment for your projects. Additionally, it frequently points you to great outside resources (like this page is doing). Most importantly, it has a very active Discord server, which fosters an aspect of community rarely found elsewhere on learning platforms.

Visit Site for More Information
freeCodeCamp()

Here is another resource that is great for learning HTML, CSS, and JavaScript. The benefit of this resource is that it is interactive, so you are given small problems to do as you learn, which help familiarize you with actual coding.

Visit Site for More Information
W3schools

Here is a resource that is great if you don’t need to necessarily practice coding but just need to learn about what things mean. It is a great supplement or reference when you can’t remember a syntax rule or two

Visit Site for More Information

For Those Who Want to Learn Data Structures and Algorithms / Problem Solving:

Princeton Algorithms

Data Structures and Algorithms course. It is very extensive, challenging, and rewarding. If you complete this, then you should be well prepared to start solving problems that show up during technical interviews.

Visit Site for More Information
Cracking the Coding Interview

If you have ever been preparing for a technical interview, then you have probably heard of this book. If not, then here you go! It is a staple when it comes to interview prep, as it covers thought and communication processes as well as core data structures and algorithms concepts needed to solve questions.

Visit Site for Book Information
Leetcode

As with Cracking the Coding Interview, you may have already heard of this if you are searching for a job or internship. If not, then this is a website where you can practice solving coding problems that will be very similar to those you will see in technical interviews. There are many other sites like Leetcode, so you can choose an alternative if you don’t enjoy the website for one reason or another

Visit Site for More Information
Coding Bat

This is another website where you can practice coding problems in Java and Python. The questions are usually much simpler than those on sites like Leetcode, but they are excellent warm-up exercises or refreshers on certain topics, such as recursion

Visit Site for More Information

For Those Who Want to Learn Math that Frequently Appears in Computer Science:

Essence of Linear Algebra

This is a YouTube series that is an amazing supplement to learning Linear Algebra in whatever setting you are in. It has great visualizations, and it really lets you appreciate how cool the subject can be in theory and in practice.

Visit YouTube channel
Essence of Calculus

Same YouTube channel as Essence of Linear Algebra--cannot recommend this resource enough.

Visit YouTube Channel
Discrete Math

This is a free online course from MIT Open Courseware. It is an in-depth course that has great lectures and also very challenging problem sets that will assure you have a good grasp of the concepts taught.

Visit Site for More Information

For Those Who Need Quick Help with Useful Tools:

Command Line
Linux Journey

This resource teaches you about the command line and much much more if you desire, not limited to text editors and networking in the Linux environment. Though comprehensive, this resource is great because it is interactive and has many sections that can be completed in a short amount of time.

Visit Site for More Information
The Linux Command Line

This is a go-to book that teaches basic commands, shell scripting, and much more. It is heavily detailed, and it is completely free to download the book as a PDF file.
